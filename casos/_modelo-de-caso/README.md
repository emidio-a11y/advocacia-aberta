# Modelo de caso

Esta é a **estrutura-modelo** de um caso. Não trabalhe aqui dentro: copie esta pasta,
renomeie para o seu caso (número do processo ou um nome curto) e trabalhe na cópia.

As subpastas (`autos/`, `analise/`, `fundamentacao/`, `pecas/`) têm um README curto
explicando o que vai em cada uma. Veja também `../README.md`.

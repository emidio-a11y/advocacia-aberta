# autos/ — o material bruto (o que ENTRA)

Coloque aqui os documentos do caso: PDFs das peças e provas, áudios/vídeos de
audiências ou reuniões, e-mails, contratos. É a matéria-prima que as skills leem.

- Áudio/vídeo? Rode `/transcrever` para virar texto.
- Pilha grande de PDFs? Rode `/organizar-caso` para gerar o resumo.
